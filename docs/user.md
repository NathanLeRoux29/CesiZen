## Premiers Pas

### Lancement de l'Application

Pour accéder à CesiZen :

1. Ouvrez votre navigateur web
2. Accédez à l'adresse de l'application
3. L'application se charge automatiquement

### Interface Utilisateur

L'application utilise un thème sombre avec des accents verts (`#04FF92`) pour une expérience visuelle apaisante et moderne.

## Navigation

### Menu Principal

La barre de navigation en haut de l'écran vous permet d'accéder aux différentes sections :

- **Accueil** : Page d'accueil avec l'article du jour
- **Catalogue de détente** : Tous nos articles

## Page d'Accueil

La page d'accueil est la première chose que vous voyez en ouvrant l'application.

### Article du Jour

En haut de la page, vous trouverez l'**article du jour**. Cet article change automatiquement chaque jour et vous propose du contenu de bien-être pertinent.

### Articles Suggérés

Sous l'article du jour, trois articles aléatoires vous sont proposés. Ces suggestions sont différentes de l'article du jour pour vous faire découvrir toujours plus de contenu.

### Bouton Catalogue

Un bouton "Découvrir le catalogue" vous permet d'accéder à la liste complète des articles.

## Connexion

### Pourquoi se Connecter ?

La connexion vous permet de :

- Personnaliser votre expérience
- Suivre vos statistiques
- Sauvegarder vos préférences
- Accéder à votre compte utilisateur

### Se Connecter

1. Cliquez sur l'icône compte en haut à droite
2. Entrez votre **email** et votre **mot de passe**
3. Cliquez sur "Se connecter"

> **Note** : Pour cette version de démonstration, vous pouvez utiliser n'importe quel email valide et un mot de passe de 6 caractères ou plus.

### Créer un Compte

La fonctionnalité de création de compte sera disponible dans une future mise à jour.

## Mon Compte

### Accéder à Mon Compte

Cliquez sur l'icône compte en haut à droite lorsque vous êtes connecté.

La page compte affiche vos statistiques personnelles :

## Exercices de Respiration

### Configuration des Séances

La page de configuration des exercices de respiration vous permet de personnaliser vos séances.

### Durée de l'Exercice

- Utilisez le curseur pour choisir la durée (1 à 30 minutes)
- La durée recommandée pour les débutants est de 5 minutes

### Cycles de Respiration

- Choisissez le nombre de cycles (1 à 20)
- Un cycle = une inspiration + une expiration

### Technique de Respiration

Plusieurs techniques sont disponibles :

- **Respiration Carrée** : 4 temps égaux (inspirer, retenir, expirer, retenir)
- **Méthode 4-7-8** : Inspiration 4s, rétention 7s, expiration 8s
- **Respiration Diaphragmatique** : Respiration profonde du ventre
- **Respiration Cohérente** : 6 cycles par minute

### Type d'Exercice

Choisissez le type d'exercice selon vos besoins :

- 🧘 **Relaxation** : Pour se détendre
- ⚡ **Énergie** : Pour se dynamiser
- 😴 **Sommeil** : Pour s'endormir
- 🧠 **Concentration** : Pour se concentrer

### Sons et Vibrations

- **Sons ambient** : Activez des sons relaxants (nature, océan, forêt, pluie)
- **Vibrations** : Activez les vibrations pour un accompagnement physique

### Sauvegarde

Cliquez sur "Sauvegarder la configuration" pour enregistrer vos préférences.

## FAQ

### L'application est-elle gratuite ?

Oui, CesiZen est entièrement gratuit pour cette version.

### Mes données sont-elles sécurisées ?

Vos données sont stockées localement sur votre appareil. Dans une future version, un système de compte sécurisé sera mis en place.

### Comment puis-je contacter l'équipe ?

Pour toute question ou suggestion, veuillez nous contacter via les canaux appropriés (à venir).

### L'application fonctionne-t-elle hors ligne ?

Certaines fonctionnalités nécessitent une connexion internet (chargement des images d'articles). La configuration des exercices de respiration fonctionne hors ligne.

### Puis-je utiliser l'application sur mobile ?

L'application est responsive et s'adapte à tous les écrans. Une version mobile via Capacitor sera disponible dans une future mise à jour.
